Первый сайт-визитка

Статус: Завершен

Технологии: HTML5, CSS3

Описание: Мой самый первый полноценный веб-проект. Сайт-визитка, который вы сейчас просматриваете. В процессе создания этого сайта я изучила основы HTML-разметки и CSS-стилей, научилась работать с семантическими тегами и создавать адаптивный дизайн.

Что я узнала: основы верстки, работу с flexbox, создание навигации, стилизацию форм и таблиц.

Шахматный таймер

Статус: В разработке

Технологии: HTML, CSS, JavaScript

Описание: Веб-приложение таймера для игры в шахматы с различными настройками времени (блиц, рапид, классика). Планируется добавить звуковые сигналы, сохранение настроек и историю партий.

Цели проекта: практика работы с JavaScript, изучение таймеров и интервалов, создание интерактивного интерфейса.

Особенность: проект сочетает моё увлечение шахматами с программированием.

Галерея цифровых рисунков

Статус: Планируется

Технологии: HTML, CSS, JavaScript, возможно React

Описание: Онлайн-галерея моих цифровых рисунков с возможностью фильтрации по категориям (портреты, пейзажи, абстракция), тегам и дате создания. Планируется система комментариев и лайков.

Что хочу изучить: работу с галереями изображений, создание фильтров, основы backend для комментариев.

Игра "Найди пару"

Статус: В планах

Технологии: HTML, CSS, JavaScript

Описание: Классическая игра на развитие памяти с тематикой аниме и игр. Разные уровни сложности, система очков, таймер и таблица рекордов.

Для кого: проект для начинающих разработчиков игр и поклонников аниме.
